# Bangalore-Restaurant-Analytics-Dashboard
📈 Data Analytics Project | Power BI • ETL • Visualization  Transformed and analyzed the Zomato Bangalore Restaurants dataset to create an interactive dashboard delivering insights into restaurant performance, customer preferences, and market trends.

## 📌 Overview

This project analyzes the **Zomato Bangalore Restaurants Dataset** using Power BI to uncover insights into restaurant ratings, cuisines, pricing, and customer preferences. An ETL process was performed to clean and transform the data before creating interactive visualizations.

## 🎯 Objectives

* Analyze restaurant ratings and performance
* Identify popular cuisines and dining trends
* Explore pricing patterns across locations
* Evaluate online ordering and table booking services
* Build an interactive business intelligence dashboard

## 🛠️ Tools Used

* Power BI
* Power Query
* Excel / CSV
* Data Cleaning & Transformation
* Data Visualization

## 📊 Dashboard Insights

* Restaurant rating analysis
* Popular cuisine trends
* Cost-for-two distribution
* Online ordering and table booking insights
* Location-wise restaurant performance

## 🔄 ETL Process

1. Extracted data from the Zomato Bangalore Restaurants dataset
2. Cleaned missing, duplicate, and inconsistent records
3. Transformed data for analysis using Power Query
4. Loaded data into Power BI for visualization
